# jostc-docker
JoSTCについては，[こちら](https://github.com/MarkMoHR/JoSTC)のリポジトリのREADMEや論文を参照されたい．

## Dockerのビルドから実行

### Docker Imageのビルド
```
cd jostc-docker/docker-shell

./build.sh
```

### Docker Imageの実行
```
cd jostc-docker/docker-shell

./run.sh
```

### 同じDocker Imageを複数個起動
```
cd jostc-docker/docker-shell

./exec.sh
```
